# light to sound
take an rgb led and pump out some bumping tunes... or you know, just tones

## requirements

* arduino
* bread board
* rgb led (common cathode)
* 3 1uF capacitor
* 1 4.7uF capacitor
* 1 k&#8486; resistor
* jumper wires
* aligator clamps
* head phones (or speakers)
*
